# AI Resume Matcher (SaaS-style, Local LLM)

End-to-end resume–job matching app with a FastAPI backend and Streamlit frontend. Users can sign up, create projects, upload a job description + multiple resumes (PDF), run matching, view ranked results with explanations, and export a consolidated PDF report — powered entirely by local Ollama models (no paid APIs).

---

## Demo (1 minute)

1) Sign up / Login  
2) Create a Project  
3) Upload Job Description (PDF) + Resumes (PDFs)  
4) Click **Run Matching**  
5) View ranked matches + download the PDF report
